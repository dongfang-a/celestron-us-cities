# Celestron NexStar US Cities KML

This repository contains a KML (Keyhole Markup Language) file that maps the coordinates of various US cities, military bases, and specific locations. The data is structured and organized for compatibility and reference with Celestron NexStar telescope systems (such as the NexStar 102SLT).

## Contents

* **`Celestron_US_Cities.kml`**: A comprehensive file containing nearly 1,000 data points across 50+ states and territories. 

## Features

* **Categorized by State**: All locations are systematically grouped into `<Folder>` tags by their respective states for easy navigation.
* **Custom Map Icons**: Includes a unified, custom star icon style for clean visualization on mapping software.
* **Alphabetical Order**: Cities within each state folder are strictly sorted in alphabetical (A-Z) order.

## Usage

This KML file can be imported into any compatible GIS or mapping software:

1.  **Google Earth (Pro / Web)**: Go to `File` > `Import` (or `Open`) and select the `.kml` file.
2.  **Google My Maps**: Create a new map, click `Import` on a new layer, and upload the file to view the locations alongside your astronomical planning.
